## Run Locally

Clone the project

```bash
   git clone git@github.com:HlaingMinThan/Github-Issues-React-Project.git github-issues
```

Go to the project directory

```bash
  cd github-issues
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

and then 

u can check on [(http://localhost:3000)](http://localhost:3000)
